Project name: Workflow Planner

Description:

Nowadays people have different work style when it comes to schoolwork, daily tasks, and work itself. The goal of this project is to provide a productivity tool that functions as a task planner. Users will be able to add tasks on board and assign each task with an estimate time to complete, or number of ‘Pomodoro’s, which we will get into in a bit. The web app also comes with a built-in Pomodoro timer

UI mock-ups:

Login Page

Users are first directed to the Workflow login screen. Login requires a username and password. New users can register for an account by selecting register.

EXTENSION: Clicking register will bring up new page that asks user for email, password and username details to make new account.

login.html

Homepage

Once logged in, users are directed to their board homepage. Here a user can add tasks to their To Do list. Users can drag tasks into different columns depending on their priority/state.

EXTENSION: Users with multiple boards can switch between boards using drop down menu. User can enable timer to monitor work

homepage.html

Settings

Users can have multiple boards which are stored in the settings tab. Users have the option to delete boards/share/goto

settings.html

Database schema

Schema

User
  • user_ID(primary key)
  • username
  • email
  • password
Board
  • board_ID(primary key)
  • description
  • user_ID (foreign)
Item
  • user_ID(primary key)
  • description
  • status(0 for to-do, 1 for in-progress, 2 for done)
  • user_ID (foreign)
  • board_ID (foreign)
  • expected pomodoros
  • completed
