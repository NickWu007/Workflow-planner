Project name: Workflow Planner

Description:

Nowadays people have different work style when it comes to schoolwork, daily tasks, and work itself. The goal of this project is to provide a productivity tool that functions as a task planner. Users will be able to add tasks on board and assign each task with an estimate time to complete, or number of ‘Pomodoro’s, which we will get into in a bit. The web app also comes with a built-in Pomodoro timer
